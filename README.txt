  OPEN_PRESENTATION.html

  index.html

It redirects to:
  cluster_intro_html/cluster_intro.html

Keep these folders together:
  cluster_intro_html
  wdelta_construction_html

Do not move cluster_intro.html alone. The construction demo slide uses
../wdelta_construction_html/index.html, so the two folders must remain siblings.

MathJax is bundled locally in:
  cluster_intro_html/mathjax/tex-svg.js

The arXiv link and QR-code target are external links, but the slide deck itself
and the embedded construction demo should open from this folder.
