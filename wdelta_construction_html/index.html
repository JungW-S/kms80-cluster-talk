Construction of WΔ(C)
𝐢=(2,3,1,2,2,1) H=(L,R,L,R,R)
Δ=(1,2,1,3,2,1)
3 -*-> 1
Step 1/3: Construct the word iΔ(C) Step 2/3: Build the weave iΔ(C) → w0 Step 3/3: Construct WΔ(C)
  • C is an admissible chain of i-boxes.
  • Δ is a reduced expression of w0.
  • First construct the word iΔ(C).
  • Place Δ in the middle.
  • Read the i-boxes in the order of C.
  • For each i-box, record its color: the index at the endpoint.
  • If c or R, then place the color on the right of Δ.
  • If L, then apply the *-involution and place the result on the left of Δ.
  • The resulting word is iΔ(C).
  • Start from the word iΔ(C).
  • For t=1,...,r, attach the strand for ct
    to the same-color Δ-strand;
    use 4-/6-valent moves if necessary;
    then create one trivalent vertex.
  • This gives a weave iΔ(C) → w0.
  • Construct another weave Δi → iΔ(C)
    using only 4- and 6-valent vertices.
  • Concatenate these two weaves.
  • This gives WΔ(C): Δi → w0.
